Descripcion del proyecto

para empezar el proyecto se llama sublimaipu,es una pyme que se de dedica a la venta de productos subliminados que son personalizados en la comuna de maipu, nuestro cliente nos ha pedido realizar un sitio web
que le permita vender sus produtos de manera online que sea perfecta para mostrar los productos que realizan y tienen en venta, con la finalidad de tener un nuevo público.

Tecnologias utilizadas

nosotros para este proyecto utilizamos las siguentes tecnologias:github,visual studio code,xampp,sqlworkbench,postman y un navegador web.

Instrucciones de instalacion

para el backend primero que todo hay que tener instalado los siguentes programas visual studio code,xampp y mysql workbench,lo segundo es abrir el proyecto en visual studio code,
lo siguente es instalar las siguentes extensiones extension pack for java y spring boot extension pack.

para el frontend primero que todo hay que tener instalado el siguente programas visual studio code.luego debemos ir al cmd verificar si java esta instalado e instalar node y sus librerias en nuestro equipo,
despues de eso hay que abrir el proyecto en visual studio code,lo ultimo es instalar las siguentes extensiones ES7 y react.

Instrucciones de ejecucion

para el backend primero que todo debemos asegurarnos de tener todos los programas y extensiones mencionados anteriormente, una vez hecho como primer paso tenemos que abrir xampp darle al boton start donde se
encuentra la fila mysql , los siguente sera abrir el mysql workbench y debemos crear una conexion aca ponemos cualquier nombre y listo,despues debemos crear una nueva base de datos con el nombre de db_sublimaipu
para esto debemos hacer click en el incono de un barril, luego se nos abre un menu ponemos apply y tendremos la base de datos creada.y por ultimo para poder correr el backend hay que hacer click en el 
boton de springboot (un boton con icono hexagonal) se despliegara un menu aca debemos hacer click en la flecha ahi se abrira la terminal y ahi en el mysql workbrench deberian crearse las tablas,si esto pasa deberia estar
todo bien y funcionando correctamente, asegurandose los beans de springboot esten en verde.

para el frontend primero que todo debemos asegurarnos de tener todas las herramientas y extensiones mencionados anteriormente, una vez hecho como primer paso tenemos que abrir este proyecto sin cerrar el backend ya que 
nada funcionara debido a que el backend manejara la base de datos.para correr el frontend debemos abrir la terminal presionando los botones control mas ñ, una vez abierto debemos escribir el comando npm run dev,despues si todo
sale bien deberia entregar un link para acceder al localhost (http://localhost:5173/) ahi hay que hacer click y se abrira una pestaña en el navegador de su eleccion y listo ahi estara abierto nuestro proyecto.
